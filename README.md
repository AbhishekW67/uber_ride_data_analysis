## 🚖 Uber Ride Data Analysis Dashboard

## 📌 Overview

This project presents an interactive dashboard analyzing Uber ride booking data. It provides insights into ride trends, revenue generation, cancellations, and customer/driver ratings to support data-driven decision-making.

---

## 🎯 Objectives

* Analyze monthly ride booking trends
* Identify top-performing vehicle categories
* Understand revenue distribution
* Investigate cancellation patterns
* Evaluate customer and driver satisfaction

---

## 🛠️ Tech Stack

* **Power BI Desktop**
* **CSV Dataset**
* Data Cleaning & Transformation

---

## 📊 Key KPIs

* **Total Rides:** 147.58K
* **Total Revenue:** ₹51.01M
* **Average Distance:** 26.01 km
* **Cancellation Rate:** 25%

---

## 📂 Dataset

The dataset includes:

* Ride bookings
* Vehicle types
* Payment methods
* Revenue
* Distance travelled
* Cancellation reasons
* Ratings

* Auto category dominates bookings and revenue
* Cancellation rate (~25%) indicates operational inefficiencies
* Customer ratings are slightly higher than driver ratings
* UPI and digital payments are widely used
